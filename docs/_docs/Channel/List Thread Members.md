# `listThreadMembers`

```php
$client->channel->listThreadMembers($parameters);
```

## Description



## Parameters


Name | Type | Required | Default
--- | --- | --- | ---
channel.id | snowflake | true | *null*

## Response

Returns array of thread members objects that are members of the thread.

Can Return:

* thread members
