# `getGuildWidgetSettings`

```php
$client->guild->getGuildWidgetSettings($parameters);
```

## Description

Requires the MANAGE_GUILD permission.

## Parameters


Name | Type | Required | Default
--- | --- | --- | ---
guild.id | snowflake | true | *null*

## Response

Returns a guild widget object.

Can Return:

* guild widget
